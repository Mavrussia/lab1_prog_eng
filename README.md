## Самостоятельная работа №1
### Дайте подробный комментарий для кода, написанного ниже. Комментарий нужен для каждой строчки кода, нужно описать что онаделает. Не забудьте, что функции комментируются по-особенному
### Код
```python
from datetime import datetime  # Импорт класса datetime для работы с датой и временем
from math import sqrt  # Импорт функции sqrt для вычисления квадратного корня


def main(**kwargs):
    """
    Функция принимает неопределенное количество именованных аргументов.
    Рассчитывает квадратный корень суммы квадратов элементов каждого переданного аргумента.
    
    :param kwargs: неограниченное количество именованных аргументов
    """
    for key in kwargs.items():  # Перебирает каждый элемент (ключ, значение) в переданных аргументах
        # Расчет квадратного корня суммы квадратов двух чисел, переданных в значении
        result = sqrt(key[1][0] ** 2 + key[1][1] ** 2)
        print(result)  # Выводит результат расчета

if __name__ == '__main__':  # Условие запуска программы только если файл запущен напрямую
    start_time = datetime.now()  # Запоминает текущее время начала выполнения программы

    # Вызов функции main с передачей пяти именованных аргументов, каждый из которых содержит список из двух чисел
    main(
        one=[10, 3],
        two=[5, 4],
        three=[15, 13],
        four=[93, 53],
        five=[133, 15]
    )

    time_costs = datetime.now() - start_time  # Рассчитывает время выполнения программы
    print(f"Время выполнения программы - {time_costs}")  # Выводит время выполнения
```

### heron.py
```python
from math import sqrt


def calculate_area(a, b, c):


    s = (a + b + c) / 2
    area = sqrt(s * (s - a) * (s - b) * (s - c))
    return area

```
